# NYSERDA Sample Project

This analysis leverages a dataset from the city of New York. Specifically, the dataset includes detailed building-level energy audit results, collected using the U.S. Department of Energy's Asset Score Audit Template (Audit Template) tool.

## Data References

NYC OpenData. 2025. “LL87 Energy Audit Data.” [https://data.cityofnewyork.us/Environment/LL87-Energy-Audit-Data/au6c-jqvf/about_data.](https://data.cityofnewyork.us/Environment/LL87-Energy-Audit-Data/au6c-jqvf/about_data)
